包括Windows、路由器、Linex(Fedora,Ubuntu)版本的拨号程序

## Windows User
1. 下载、解压
2. 建议一个新的网络连接，取名为test_ras
2. 进入解压后的路径
3. 双击运行dial.exe

第一次运行，需要输入用户名和密码。建议在cmd中切换到解压后的路径中执行dial.exe，当登陆成功后显示“you win”。
